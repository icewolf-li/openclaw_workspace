# GitHub 知识库技能

将任何 GitHub 仓库转换为结构化的本地知识库，支持智能问答、代码检索和文档分析。

## 功能特点

- 📥 自动克隆 GitHub 仓库（公开/私有）
- 📚 提取并索引 README、文档、代码注释
- 🔍 支持自然语言问答（"这个项目如何安装？"）
- 💡 代码理解（"解释 main.py 的功能"）
- 🗂️ 自动生成知识库摘要

## 使用方法

### 基本用法
```
将 [仓库URL] 转换为知识库
从 GitHub [用户名/仓库名] 创建本地知识库
```

### 高级选项
- **私有仓库**：提供访问令牌
- **特定分支**：指定分支名称
- **文件过滤**：只处理特定文件类型（.md, .py, .js 等）

## 技术实现

使用以下工具链：
- `git` - 仓库克隆
- `tree-sitter` - 代码解析
- 向量嵌入 - 语义搜索
- 本地存储 - 知识库持久化

## 示例对话

用户：「把 facebook/react 转成知识库」
AI：✅ 克隆完成！已索引 1248 个文件，包括文档、源码和测试。现在可以问：
- "React 的核心概念是什么？"
- "show me the useState hook implementation"
- "这个项目有哪些主要组件？"

## 注意事项

- 首次处理大型仓库可能需要几分钟
- 私有仓库需要 GitHub Personal Access Token
- 知识库存储在 `~/.openclaw/knowledge/` 目录
